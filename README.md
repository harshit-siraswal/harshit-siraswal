### 🚀 StudyShare *(Private — coming public soon)*

A full-scale educational platform for college students to share resources, track attendance, and collaborate.

| Layer | Tech | Description |
|---|---|---|
| 📱 **Mobile App** | Flutter · Firebase · Dart | Cross-platform app for Android & iOS. AI chat, syllabus viewer, YouTube integration, push notifications, KIET attendance integration |
| 🖥️ **Admin Panel** | JavaScript · HTML · CSS · Node.js | Web dashboard for admins. Manages users, notices, resources, reports, push notifications, and ban control |
| ⚙️ **Backend API** | Node.js · TypeScript · PostgreSQL · Redis · Supabase · Firebase | Secure proxy with auth (Firebase), RBAC, rate limiting, follow/bookmark system, Docker + EC2 + Render deployment, GitHub Actions CI/CD |

**Key features:** AI NotebookLM integration, n8n automation workflows, RAG v2 document search, reCAPTCHA, WebSocket support.
